## Bendego
ECサイトのデータを作成するためのアプリ


# 開発環境
・eclipse2024-03
・pycharm
・java: 21
・python: 3.10+


# 機能
・商品ごとにフォルダ分け
・作成したデータのコンタクトシートを作成
・作成したデータの数を計測
・EXCELデータから行数を計測
・Excelデータの特定のシートをPDFに変換
・画像をセグメントし、背景を削除する
・セグメンテーションAIモデルを学習する


# インポート

・numpy
・torch
・pillow
・apache
・cv2
・matplotlib
